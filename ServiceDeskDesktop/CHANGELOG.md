# Changelog

Все заметные изменения в проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] — 2026-06-09

### Добавлено
- Окно первого запуска для создания учётной записи администратора.
- Окно входа с проверкой роли через Firebase Auth.
- Главное окно диспетчера с таблицей заявок, поиском и фильтрацией.
- Окно создания и редактирования заявок с валидацией полей.
- Назначение полевого инженера из списка пользователей Firebase.
- Внутренние и внешние комментарии, метки диспетчера.
- Экспорт заявок в Excel и Word с группировкой по статусам.
- Окно аналитики со столбчатой диаграммой.
- Панель администратора: создание, блокировка, сброс пароля, удаление пользователей.
- Синхронизация с Firebase Firestore, офлайн-очередь, стратегия LWW.
- Логирование операций в файл.
- Переключение светлой и тёмной темы.
- Кнопка выхода с подтверждением.

### Исправлено
- Устранено дублирование заявок при создании.
- Исправлена ошибка NullReferenceException при удалении заявки без DocumentId.
- Исправлено отображение даты создания заявки.
- Оптимизирована загрузка данных в DataGrid.
